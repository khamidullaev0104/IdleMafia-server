# GangWarsAI server

----

### Config
Create local config file (.env) coping example.env

``
cp example.env .env
``

Take a look at .env ans update all variables

### Launch server
To start dev server call start command

``
npm run start
``

### Checks
To check code by eslint 

``
npm run lint
``

To reformat code 

``
npm run format
``
